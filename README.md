# Краткое описание файлов


* `path1.py` : файл с извлечением сущночтей из запроса пользователя (Deepseek)
* `path2.py` : файл с нахождением узлов в Memgraph
* `path3.py` : файл с обходом графа, выводом подграфа в текстовом виде (для LLM)
* `path4.py` : файл с обходом векторной бд
* `main.py` : файл, запускающий весь процесс - выводит ответ нейронки
